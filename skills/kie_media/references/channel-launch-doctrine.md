## Monetization Paths (YouTube 2026)

| Path | Threshold |
|---|---|
| Long-form | 1,000 subscribers + 4,000 watch hours |
| Shorts | 1,000 subscribers + 10M Shorts views in 90 days |

Studio default: **Shorts path** (faster testing, faster virality signal, lower production cost per asset).

---

## 5 Decisions Every Channel Launch Must Answer

Before posting day one, answer all five. Leaving any unanswered = founder judgment drain on every subsequent post.

1. **Niche:** demand-proven, not passion-picked. Ask "what is already working?" not "what do I like?" Validated signals: view velocity on existing channels, RPM range, repeatable content surface.
2. **Daily volume:** 3-5 Shorts per day at launch. Not random. Scheduled. Treat as a system input, not a creative decision.
3. **Content formula:** use the 4-part structure below for every Short. No improvisation until data shows what to evolve.
4. **Monetization path:** decide upfront. Shorts path (10M views/90 days) or long-form path (4K watch hours). Different content shapes, different pacing.
5. **Pivot rule:** once a format performs, replicate it identically (same structure, different topic) before evolving. Copy proven, replicate winners, evolve later.

---

## 4-Part Content Formula (Every Short)

| Part | Timing | Purpose |
|---|---|---|
| Hook | 0-2 sec | Stop the scroll. "This tool feels illegal to know..." |
| Value | 3-20 sec | Useful, surprising, or shocking. Deliver immediately. |
| Curiosity Gap | before end | Leave something slightly incomplete. Triggers rewatch or follow. |
| Loop Ending | final frame | Make the video loop-worthy without the viewer realizing. |

Apply this structure to every Short, every channel. Deviation requires a data reason.

---

## Iteration Protocol

Track these three metrics per video, every video:

- **Retention rate:** if low, hook is weak
- **Average watch duration:** if low, pacing or value is weak
- **Drop-off point:** identifies exactly where interest dies

Every video makes the next one better. "Post and hope" is not a protocol.

---

## Subscriber Conversion Tactics

Views are not the goal. Subscribers are.

- No generic CTAs ("subscribe for more")
- Create curiosity-based series (part 2 tension, open loops)
- Make content that forces "I need to follow this account" rather than asking for it

## Common Mistakes (Anti-patterns)

| Mistake | Fix |
|---|---|
| Passion niche over proven niche | Research demand first |
| Posting without a data loop | Set up retention/watch-duration tracking before first post |
| Chasing one viral video | Pattern replication > lottery-ticket thinking |
| Being "unique too early" | Copy proven formats first; evolve with data |
| Quitting before algorithm trust builds | Momentum is non-linear; trust compounds after Day 5-6 |
