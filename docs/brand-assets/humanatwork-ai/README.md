## Design system

| Element | Spec |
|---|---|
| **Typography** | Fraunces serif (Google Fonts). "Human" = italic, "at Work" = roman. |
| **Accent mark** | Brick-red period (·) at end of wordmark. Color `#A8341F`. Reads "we are done talking, here is the work." |
| **Symbol motif** | 3 horizontal lines + 1 brick-red dot. Reads as "ledger lines of work + human anchor." NOT a letter mark. |
| **Color palette** | Paper `#F4EFE6` (Conakry-clay) / Ink `#161513` / Brick `#A8341F` / Sage `#3E4A2E` / Bone `#ECE5D6` |

## Brand-mark rules (per `feedback_no_draft_or_letter_brand_marks` memory)

- **NEVER** use a single-letter brand-mark block (e.g., a `[H]` or `[HW]` box next to the wordmark). Banned.
- **NEVER** ship pages with "DRAFT" or "PLACEHOLDER" badges on production.
- Wordmark + accent OR actual designed mark. No letter blocks.
