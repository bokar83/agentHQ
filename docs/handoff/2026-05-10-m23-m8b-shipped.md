- **Subagent branch discipline:** Always state `git checkout <branch>` as a step in subagent prompts. Tasks 2+3 committed to main instead of feat branch — had to cherry-pick. See `feedback_subagent_branch_discipline.md`.
- **Shared remote complexity:** agentsHQ and agentsHQ-echo share `bokar83/agentHQ.git`. Frontend merge conflicts are common when other sessions push to same remote. Resolve by direct edit on main, not cherry-pick across diverged branches.
- **M6 retroactive fix pattern:** When a milestone is live but roadmap stale, fix the roadmap immediately — don't wait for a "cleanup session." The stale milestone blocked M8b + M23 audit trail for weeks.
