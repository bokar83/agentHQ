The kit breaks into 8 modules, each 15-60 minutes. You can complete it across 3-4 evenings if you cannot find 24 contiguous hours. The 30-day plan in module 06 starts from whatever pace you actually have.
Q3Does it work for ChatGPT users, not just Claude?
Yes. The structure is identical. The buttons are in slightly different places. I note the differences as we go. Use the tool you already have an account for, not the one you "should" be using.

The principles (how to think about AI workflow, how to write prompts, how to set up Projects) are stable. The tool-specific buttons change. Updates are free for all buyers as the landscape shifts. I ship a revision every time Claude or ChatGPT does a major release.
