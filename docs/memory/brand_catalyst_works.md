# Catalyst Works Brand Colors

## System Overview

4 palettes, each tied to a specific context. Use the right palette for the right output — this controls perception, not just aesthetics.

**80/20 rule:** Default to Coastal Clarity (Palette 1). Use others purposefully.

---

## Non-Negotiables

- No red tones
- Do NOT use old teal `#005E60` or lime `#C8FF08`
- Orange always accent only (≤10%)
- Clay `#B47C57` = human signature color across ALL palettes

---

## 1. Coastal Clarity — DEFAULT

**Use for:** Website, LinkedIn posts/carousels, sales pages, client deliverables, revenue-facing content
**Why:** Clean, premium, modern → builds trust fast (exec audience)

| Name | Hex | Role |
|---|---|---|
| Ocean Cyan | `#00B7C2` | Primary |
| Midnight Navy | `#071A2E` | Base / background |
| Orange | `#FF7A00` | Accent CTA (≤10%) |
| Clay | `#B47C57` | Human warmth |
| Mist | `#F3F6F9` | Light background |
| Carbon | `#1E222A` | Text / contrast |

---

## 2. Baobab Nightfall — STORY / IDENTITY

**Use for:** Keynotes, thought leadership, personal storytelling, cultural narratives (Africa, leadership, philosophy)
**Why:** Depth, richness, emotional connection

| Name | Hex | Role |
|---|---|---|
| Night Indigo | `#14213D` | Base |
| Sand Clay | `#C18E63` | Warmth |
| Lagoon | `#2BD9C5` | Accent |
| Palm Green | `#3F7D3B` | Secondary |
| Dune | `#F5F1E8` | Light background |
| Sable | `#1B1E22` | Text / contrast |
