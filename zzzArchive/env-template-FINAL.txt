# ── AI MODELS ────────────────────────────────────────────────
# OpenRouter — unified gateway for all AI models
# Get from: openrouter.ai > Keys > Create Key

# ── AGENT WEB SEARCH ─────────────────────────────────────────

# Get from: serper.dev > Sign up free > API Key

# ── SECURITY REPORT EMAIL ────────────────────────────────────
# The email address where your daily security report gets sent
# This is your email — not a password

# ── YOUR WHATSAPP NUMBER ─────────────────────────────────────
# Your PERSONAL WhatsApp number in international format
# Used by the Saturday Orchestrator and security alerts
# Example: 15551234567@c.us (US number, no + sign, no spaces)

# ── GITHUB (for Always Save) ─────────────────────────────────
# Personal Access Token — classic

#           Personal access tokens > Tokens (classic)

# Your GitHub username (not email — the @handle)

# The repo where agent outputs get saved
# Create this repo at github.com first (can be private)
